C++ Programming Cheatsheet

Basic Syntax

  • #include <iostream> - Include I/O stream library.
  • cout << "Hello World"; - Output to console.
  • int main() - Main function.
  • cin >> num; - Input from user.
  • class ClassName { } - Define class.
  • public, private, protected - Access specifiers.
  • this - Pointer to current object.
  • new, delete - Dynamic memory management.
  • virtual function() - Virtual function for polymorphism.
  • inheritance - Extending classes.
  • namespace std; - Namespace declaration.
  • friend function() - Friend function declaration.
  • try, catch - Exception handling.
  • vector - C++ standard library container.
  • template<typename T> - Template function or class.
