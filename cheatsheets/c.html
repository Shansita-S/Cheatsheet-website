C Programming Cheatsheet

Basic Syntax

  • #include <stdio.h> - Include standard I/O library.
  • printf("Hello, World!"); - Print message to console.
  • int a = 10; - Declare an integer variable.
  • scanf("%d", &a); - Read user input.
  • for (int i = 0; i < 10; i++) - Looping structure.
  • if (condition) - If statement.
  • else - Else statement.
  • switch(x) - Switch case structure.
  • return 0; - Return from main function.
  • void function_name() { } - Function definition.
  • char c = 'a'; - Character variable.
  • double pi = 3.14; - Double precision variable.
  • const int MAX = 100; - Define constant value.
  • int arr[10]; - Declare an array.
  • struct Person { } - Structure definition.
