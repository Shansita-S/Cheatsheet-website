CSS Cheatsheet

Selectors

  • element { } - Element selector.
  • .class { } - Class selector.
  • #id { } - ID selector.
  • * { } - Universal selector.
  • element, element - Grouping selectors.
  • element:hover { } - Pseudo-class selector.
  • element::after { } - Pseudo-element selector.

Box Model

  • width, height - Set element size.
  • padding - Space inside the border.
  • margin - Space outside the border.
  • border - Border of the element.

Layout

  • display: block; - Block element.
  • display: inline; - Inline element.
  • display: flex; - Flexbox layout.
  • display: grid; - Grid layout.

Positioning

  • position: static; - Default positioning.
  • position: relative; - Relative positioning.
  • position: absolute; - Absolute positioning.
  • position: fixed; - Fixed positioning.

Typography

  • font-size - Set font size.
  • font-family - Set font family.
  • color - Set text color.
  • line-height - Line height.
  • letter-spacing - Adjust spacing between letters.
