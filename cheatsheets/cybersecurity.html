Cybersecurity Cheatsheet

Key concepts and practices to secure systems and data.

Cybersecurity Concepts

  • Encryption - The process of converting data into a secret code.
  • Hashing - Converting data into a fixed-size string of characters, typically used for password storage.
  • Phishing - A form of cyberattack that tricks individuals into revealing sensitive information.
  • SQL Injection - A code injection technique that exploits vulnerabilities in a database.
  • Cross-Site Scripting (XSS) - A security vulnerability that allows attackers to inject malicious scripts into web pages.
  • Cross-Site Request Forgery (CSRF) - A malicious attack that forces a user to perform unwanted actions.
  • Firewall - A security system that monitors and controls incoming and outgoing network traffic.
  • Two-Factor Authentication (2FA) - A security process that requires two forms of identification to access a system.
  • Malware - Software designed to harm or exploit a computer system.
  • VPN - A virtual private network used to secure internet connections and protect privacy.
  • Access Control - Limiting access to systems, resources, and information based on policies.
  • Public Key Infrastructure (PKI) - A framework for securing communications over the internet using encryption keys.
  • Ransomware - Malicious software that locks a user out of their files and demands payment for access.
  • Security Patch - Updates to software that fix vulnerabilities or weaknesses.
  • Zero-Day Attack - An attack that targets vulnerabilities before the vendor can release a patch.
