Data Analytics Cheatsheet

Important data analytics techniques and concepts.

Data Analytics Concepts

  • Data Cleaning - Removing or correcting erroneous or incomplete data.
  • Data Transformation - Converting data into the desired format or structure.
  • Exploratory Data Analysis (EDA) - Summarizing the main characteristics of a dataset.
  • Statistical Analysis - Using mathematical tools to interpret data and make predictions.
  • Hypothesis Testing - A method to test assumptions about a population based on sample data.
  • Regression Analysis - Modeling the relationship between variables to predict future data.
  • Machine Learning - Using algorithms to analyze data and make predictions or decisions.
  • Time Series Analysis - Analyzing data that is indexed by time.
  • Clustering - Grouping similar data points together.
  • Classification - Categorizing data into predefined classes.
  • Dimensionality Reduction - Reducing the number of input variables in a dataset.
  • Data Visualization - Representing data in graphical form to identify patterns.
  • Correlation Analysis - Analyzing the relationship between two variables.
  • Data Warehousing - Storing and managing large datasets for analysis.
  • Big Data - Handling datasets that are too large to be processed with traditional data processing tools.
