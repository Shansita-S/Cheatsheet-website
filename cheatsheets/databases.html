Database Concepts

  • SELECT - Retrieve data from a database.
  • INSERT - Insert data into a table.
  • UPDATE - Modify existing data in a table.
  • DELETE - Remove data from a table.
  • JOIN - Combine data from multiple tables.
  • WHERE - Filter records based on specific conditions.
  • GROUP BY - Group rows that have the same values into summary rows.
  • ORDER BY - Sort the result set.
  • HAVING - Filter groups after applying GROUP BY.
  • DISTINCT - Select unique values.
  • LIMIT - Restrict the number of rows returned.
  • INDEX - Improve the speed of database queries.
  • PRIMARY KEY - A unique identifier for records in a table.
  • FOREIGN KEY - A field in one table that links to the primary key in another table.
  • Normalization - Organizing a database to reduce redundancy and dependency.
