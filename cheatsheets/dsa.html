DSA Cheatsheet

Key data structures and algorithms you should know.

Core DSA Concepts

  • Array - A collection of elements identified by index.
  • Linked List - A linear collection of elements where each element points to the next.
  • Stack - A linear data structure that follows LIFO (Last In First Out).
  • Queue - A linear data structure that follows FIFO (First In First Out).
  • Tree - A hierarchical data structure with nodes connected by edges.
  • Graph - A collection of nodes (vertices) and edges connecting them.
  • Hash Table - A data structure that maps keys to values.
  • Heap - A binary tree-based data structure used for priority queues.
  • Sorting Algorithms - Techniques to arrange data in a specific order (e.g., Quick Sort, Merge Sort).
  • Binary Search - An efficient searching algorithm for sorted data.
  • Recursion - A function that calls itself to solve a problem.
  • Graph Traversal - Techniques for visiting all nodes in a graph (e.g., DFS, BFS).
  • Dynamic Programming - A method for solving complex problems by breaking them down into simpler subproblems.
  • Greedy Algorithm - An approach for making optimal choices at each step in problem-solving.
  • Divide and Conquer - An algorithm design paradigm based on breaking a problem into smaller subproblems.
