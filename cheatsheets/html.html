HTML Cheatsheet

Basic Tags

  • <html> - Defines HTML document.
  • <head> - Contains metadata about the page.
  • <body> - Body content of the page.
  • <h1> to <h6> - Header tags.
  • <p> - Paragraph tag.
  • <a href="url"> - Anchor tag for links.
  • <img src="image.jpg"> - Image tag.
  • <ul> - Unordered list.
  • <ol> - Ordered list.
  • <li> - List item.
  • <div> - Division for layout.
  • <span> - Inline element.
  • <form> - Form tag for input elements.
  • <input> - Input field for forms.
  • <label> - Label for form elements.
