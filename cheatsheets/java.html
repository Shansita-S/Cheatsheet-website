Java Cheatsheet

Basic Syntax

  • int num = 10; - Declare an integer variable.
  • public static void main(String[] args) { } - Main method.
  • if (condition) - If statement.
  • else - Else statement.
  • for (int i = 0; i < 10; i++) - For loop.
  • while (condition) - While loop.
  • try { } catch { } - Exception handling.
  • public class MyClass { } - Class declaration.
  • new MyClass(); - Object instantiation.
  • String str = "Hello"; - String declaration.
  • System.out.println() - Output to console.
  • int[] arr = {1, 2, 3}; - Array initialization.
  • import java.util.ArrayList; - Import statement.
  • List<String> list = new ArrayList<>(); - List initialization.
  • super(); - Call superclass constructor.
