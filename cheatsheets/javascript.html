JavaScript Cheatsheet

Basic Syntax

  • let variable = 10; - Declare a variable.
  • const variable = 20; - Declare a constant.
  • function myFunction() { } - Function declaration.
  • return; - Return statement from function.
  • if (condition) { } - If statement.
  • else { } - Else statement.
  • for (let i = 0; i < 10; i++) { } - For loop.
  • while (condition) { } - While loop.
  • document.getElementById("id").innerHTML = "Text"; - Modify DOM.
  • JSON.parse(jsonString); - Parse JSON.
  • JSON.stringify(object); - Convert object to JSON.
  • async function fetchData() { } - Asynchronous function.
  • await fetch(url); - Wait for fetch response.
  • map(), filter(), reduce() - Array methods.
  • setTimeout(function() { }, 1000); - Timeout function.
