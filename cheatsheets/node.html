Node.js Cheatsheet

Basic Syntax

  • const http = require('http'); - Import HTTP module.
  • const fs = require('fs'); - Import FileSystem module.
  • http.createServer((req, res) => { }); - Create HTTP server.
  • fs.readFile('file.txt', 'utf8', (err, data) => { }); - Read file asynchronously.
  • const app = express(); - Create Express app.
  • app.listen(3000, () => { }); - Start server on port 3000.
  • require('module'); - Import other modules.
  • process.argv - Access command line arguments.
  • setTimeout(function() { }, 1000); - Timeout function.
  • setInterval(function() { }, 1000); - Repeated function call.
  • JSON.parse(string); - Parse JSON string.
  • JSON.stringify(object); - Convert object to JSON.
  • require('dotenv').config(); - Load environment variables.
  • app.use(express.json()); - Middleware to parse JSON requests.
  • module.exports = functionName; - Export function for reuse.
