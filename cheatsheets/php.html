PHP Cheatsheet

Basic Syntax

  • <?php - Start PHP block.
  • echo "Hello World"; - Output text to the browser.
  • $variable = 10; - Declare a variable.
  • if ($condition) { } - If statement.
  • else { } - Else statement.
  • for ($i = 0; $i < 10; $i++) { } - For loop.
  • while ($condition) { } - While loop.
  • function function_name() { } - Function declaration.
  • return $value; - Return from function.
  • $array = array(1, 2, 3); - Array initialization.
  • $_POST['key'] - Access form data (POST).
  • $_GET['key'] - Access form data (GET).
  • include('file.php'); - Include a PHP file.
  • require('file.php'); - Require a PHP file.
  • isset($variable) - Check if variable is set.
