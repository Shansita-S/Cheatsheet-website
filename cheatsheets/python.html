Python Cheatsheet

Basic Syntax

  • variable = 10 - Declare a variable.
  • def function_name(): - Function declaration.
  • return value - Return statement.
  • if condition: - If statement.
  • else: - Else statement.
  • for i in range(10): - For loop.
  • while condition: - While loop.
  • import module - Import a module.
  • print("Hello, World!") - Print function.
  • list = [1, 2, 3] - List declaration.
  • dict = {"key": "value"} - Dictionary declaration.
  • lambda x: x + 1 - Lambda function.
  • try: except: finally: - Exception handling.
  • with open('file.txt') as file: - File handling.
  • list.append(value) - Append to list.
